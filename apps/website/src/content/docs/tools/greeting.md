**Greeting** is a demo / smoke-test tool. It exists so you can confirm a brand-new
MCP server is connected and responding before you add anything real — it has no
side effects.

## Tool

- **Greeting** — returns a templated hello for a given name, in English or Spanish.
  It's meant only for testing (e.g. asking your client to "test the greeting
  tool"); in normal conversation the assistant just replies directly rather than
  calling it.

Once you've confirmed the server works, move on to a real integration like
[Gmail](/docs/tools/gmail), [Slack](/docs/tools/slack), or
[Web Search](/docs/tools/web-search).
